## 推荐阅读方式

如果你想快速建立短剧编剧框架，可以按这条线读：

1. **01**：先理解短剧行业和新人入行门槛。
2. **02-04**：看题材/IP、黄金开场、主线设计，解决“故事怎么立住”。
3. **05-07**：看节奏、悬念、钩子、情绪点，解决“观众为什么继续看”。
4. **08-10**：看剧本落地性、台词打磨、网文作者转型，解决“稿子怎么更容易过稿和拍出来”。
