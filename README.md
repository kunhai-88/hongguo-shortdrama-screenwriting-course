# 红果官方短剧编剧课整理：短剧 / 漫剧脚本创作手册

这不是一份普通的「编剧资料合集」。

它更像一份从平台和工作室视角拆出来的短剧/漫剧审稿清单：一个故事为什么值得做，第一集为什么能抓住人，主线为什么撑得住，观众为什么会追，制作方为什么敢拍。

短剧和漫剧最容易卡住的地方，往往不是「没有好点子」。

真正卡住的是：你不知道平台、制作方、工作室到底拿什么标准判断一个稿子。

这份仓库整理的是「红果短剧创作服务平台」公众号《短剧编剧第一课》01-10 期。它适合写 **短剧剧本、漫剧剧本、AI 漫剧分镜脚本、网文改短剧大纲** 的人放在手边反复查。

一句话说：它不是教你把故事写得更文学，而是教你把一个想法改成平台愿意选、制作方敢拍、观众愿意追的稿子。

## 为什么这套课值得学

这套课有价值，不是因为它叫「编剧课」，而是因为它的来源和视角都很稀缺。

很多外部教程会讲技巧，但缺少行业坐标。红果这套课把选题、开场、主线、节奏、悬念、情绪点、过稿、拍摄、台词放到同一个现实系统里讲：不是「我觉得好看」，而是「这个稿子为什么能被选中、被拍出来、被观众继续看」。

**1. 它是红果官方发出来的。**

短剧行业里，很多经验是口口相传的：平台喜欢什么、什么稿子容易被退、开场为什么不抓人、制作方为什么不敢拍。红果这套课把其中一部分标准公开讲了出来。

这比外部猜测更有参考价值。

**2. 讲课的人不是纯理论老师，而是一线工作室。**

10 期里出现了凉白开、白羊、刀锋、咪呜、骏乐、燕北等平台入驻或爆款工作室。他们讲的不是「故事要有起承转合」这种大话，而是更具体的问题：

- 为什么有的题材看起来热，但稿子不好过；
- 为什么第一集第一场就能决定一个剧本有没有机会；
- 为什么主线模糊，后面再多高能桥段也救不回来；
- 为什么台词不能只是好听，还要推动剧情；
- 为什么一个 100 分创意，如果拍不出来，也不如一个能执行的 80 分剧本。

**3. 它讲的是平台和制作的现实，不只是写作审美。**

短剧/漫剧不是只看「故事好不好」。它还要同时回答：

- 观众第一眼为什么停下来；
- 观众下一集为什么还要看；
- 平台为什么觉得这个题材有机会；
- 制作方为什么觉得这个场景能拍；
- 演员能不能演；
- 台词能不能被听懂；
- 前三集能不能立住主线。

这些问题，才是短剧和漫剧真正的门槛。

## 这份资料适合怎么用

如果你已经有一个故事，可以把它当成一份审稿清单：

- 题材有没有稳定的情绪需求？
- 第一集有没有快速建立困境、冲突、反转或期待？
- 主角的目标能不能一句话讲清楚？
- 每集有没有信息、关系、局势或情绪变化？
- 结尾有没有未完成感？
- 台词是在推动剧情，还是只是在解释设定？
- 这个场景导演能不能拍、演员能不能演、制作方能不能接受？

如果这些问题答不上来，剧本大概率还没到能交稿的状态。

## 对短剧和漫剧都有什么用

短剧和漫剧的表现形式不同，但创作压力很像：

- 都需要低理解成本；
- 都需要强开场；
- 都需要连续追看理由；
- 都怕台词解释太多；
- 都怕一集过去，关系和局势没有变化；
- 都需要把抽象情绪变成观众看得见的动作、冲突和选择。

所以这套课不只适合短剧编剧，也适合做漫剧、AI 漫剧、动态漫、剧情号的人。

尤其是做漫剧时，03、05、06、07、08、09 这几期很值得反复看：开场、节奏、悬念、情绪点、可拍性、台词，基本就是漫剧脚本最容易出问题的地方。

## 推荐阅读路线

**如果你是新手：**

先读 01，再按 02-10 顺序读一遍。不要一开始就只看技巧。短剧的核心不是文笔，而是市场判断、情绪设计和可执行表达。

**如果你已经有一个故事：**

按这个顺序查：

1. 02：题材/IP 有没有用户需求；
2. 03：第一集开场能不能抓人；
3. 04：主线目标是否清晰；
4. 05-07：每集是否有节奏、悬念和情绪点；
5. 08：场景、预算、镜头、审核是否能过；
6. 09：台词是否能演、能听懂、能推动剧情。

**如果你在做漫剧或 AI 漫剧：**

重点读 03、05、06、07、08、09。漫剧更依赖画面表达和连续追看，最怕开场慢、动作不可视、台词解释过多、每集没有变化。
