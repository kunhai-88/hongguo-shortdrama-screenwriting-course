这是「红果短剧创作服务平台」公众号《短剧编剧第一课》01-10 期的学习整理版。

如果你正在写 **短剧剧本、漫剧剧本、AI 漫剧分镜脚本、网文改短剧大纲**，这套资料很适合放在手边反复查。它不是泛泛讲“怎么写故事”，而是把短剧创作里最现实的几个问题拆开讲清楚：

- 选题怎么判断有没有市场；

- 主线怎么撑起 60-100 分钟内容；

- 剧本怎么写得更容易过稿、拍出来；
- 网文作者怎么转成短剧/漫剧编剧。

## 这套课的核心价值

这 10 期可以看成一条短剧/漫剧剧本生产线：

```text
入行认知
  -> 题材/IP 判断
  -> 黄金开场
  -> 主线设计
  -> 节奏控制
  -> 悬念、期待、钩子
  -> 情绪点设计
  -> 过稿与落地
  -> 台词打磨
  -> 网文作者转型
```

它真正有价值的地方在于：不是只讲“故事要好看”，而是把短剧这种内容产品的约束讲得很具体。

- 观众为什么愿意点开；
- 观众为什么愿意继续看；
- 平台为什么觉得这个题材有机会；
- 制作方为什么觉得这个剧本能拍；
- 编剧为什么不能只按小说/长剧习惯写；
- 每一集怎么让关系、信息、情绪或局势发生变化。

对短剧有用，对漫剧也有用。因为漫剧虽然表现形式不同，但同样依赖 **强开场、强情绪、强钩子、低理解成本、连续追看理由**。

## 推荐阅读路线

**如果你是新手：**

先读 01，再按 02-10 顺序读一遍。不要一开始就只看技巧，因为短剧的核心不是“文笔”，而是市场判断、情绪设计和可执行表达。

**如果你已经有一个故事：**

按这个顺序查：

1. 02：题材/IP 有没有用户需求；
2. 03：第一集开场能不能抓人；
3. 04：主线目标是否清晰；
4. 05-07：每集是否有节奏、悬念和情绪点；
5. 08：场景、预算、镜头、审核是否能落地；
6. 09：台词是否能演、能听懂、能推动剧情。

**如果你在做漫剧或 AI 漫剧：**

重点读 03、05、06、07、08、09。漫剧更依赖画面表达和连续追看，最怕开场慢、动作不可视、台词解释过多、每集没有变化。

## 我建议收藏这份资料的原因

很多编剧教程会停在“人物要立住、故事要好看、情绪要强”这种正确但不够落地的层面。

这套红果官方编辑课更接近一线项目经验：它反复强调 **市场、平台、制作、观众** 四个约束。对真正要写短剧/漫剧的人来说，这比抽象创作鸡汤更有用。

你可以把它当成一份检查清单：

- 这个题材有没有稳定情绪需求？
- 开场有没有迅速建立困境、冲突、反转或期待？
- 主线是不是一句话能讲清楚？
- 每集有没有信息、关系、局势或情绪变化？
- 结尾有没有未完成感？
- 台词是不是在推动剧情，而不是解释作者想法？
- 这个场景导演能不能拍、演员能不能演、制作方能不能接受？

如果这些问题答不上来，剧本大概率还没到能交稿的状态。
